# Software-Based Return-to-Sport Assessment Using Medical Imaging and Motion Analysis

## Overview

This project presents a multimodal clinical decision-support framework for assessing Return-to-Sport (RTS) readiness following ACL injury.

The system combines:

- MRI-based ACL structural assessment
- Biomechanical movement analysis
- Decision-level fusion
- Explainable AI (Attention + Grad-CAM)
- Web-based clinical decision support platform

## Technologies

- Python
- PyTorch
- XGBoost
- FastAPI
- React
- Medical Imaging
- Machine Learning

## Results

### MRI Assessment
- Mean Cross-Validation AUC: 0.917
- Best AUC: 0.922

### Biomechanical Assessment
- RTS Classification Accuracy: 95.2%
- Average Confidence: 95.4%
